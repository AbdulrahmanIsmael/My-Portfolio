# Portfolio Website

A personal portfolio website built with Next.js, React, and TypeScript. Includes bilingual support (English/Arabic), dark/light mode, animations, and 3D graphics.

## Features

- Bilingual support (Arabic & English) using next-intl
- Dark/light theme toggle
- Smooth animations with Motion library
- 3D graphics with OGL
- Responsive design
- Contact form with EmailJS
- TypeScript support

## Tech Stack

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
- Motion (animations)
- OGL (3D graphics)
- next-intl (i18n)
- Zustand (state management)
- React Hook Form
- EmailJS

## Project Structure

```
app/              # Next.js pages and API routes
components/       # React components
  ├── header/
  ├── sections/  # Main page sections
  ├── ui/        # Reusable UI components
i18n/             # Internationalization config
layout/           # Layout components
lib/              # Utilities and constants
messages/         # Translation files
public/           # Static assets
stores/           # Zustand state management
styles/           # Global styles
types/            # TypeScript types
```
